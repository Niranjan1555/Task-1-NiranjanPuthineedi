# Rule-Based AI Chatbot

## Description
This project is a simple rule-based AI chatbot developed using Python as part of the DecodeLabs Industrial Training Program (Week 1). The chatbot responds to predefined user inputs using if-elif-else statements and runs continuously until the user exits.

## Features
- Greetings and exit commands
- Date and time responses
- College-related questions
- Joke responses
- Calculator operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Help command
- Default response for unknown inputs
- Continuous interaction using a while loop

## Technologies Used
- Python 3
- if-elif-else statements
- while loop
- datetime module
